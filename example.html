Method 2: Server-Side PNG Generation

Use the Node.js server to generate PNGs on-the-fly via URL parameters.

Start the Server

npm install
npm start

Server runs on http://localhost:3000

Use as Image URL

<img
    src="http://localhost:3000/png?data=16x16:000...&scale=8"
    alt="Pixel art sprite"
/>

Example with actual server (if running):

Start the server with npm start to see this:

URL Parameters

  • data - The pixel art string (required)
  • scale - Scale multiplier 1-16 (default: 1)
  • format - Output format: png or svg (default: png)
‚úì Advantages:
  • Works with standard <img> tags
  • Server-side rendering
  • Cacheable images
  • SVG fallback option
  • No client-side JavaScript needed

Advanced Examples

CSS Styling

inline-px {
    display: inline-block;
    filter: drop-shadow(2px 2px 4px rgba(0,0,0,0.5));
    transition: transform 0.2s;
}

inline-px:hover {
    transform: scale(1.1);
}

JavaScript API

// Create element dynamically
const sprite = document.createElement('inline-px');
sprite.setAttribute('data', '16x16:000...');
sprite.setAttribute('scale', '4');
document.body.appendChild(sprite);

// Update dynamically
sprite.setAttribute('scale', '8');
